# Future Work

The current system provides a solid foundation for detecting potential threats in artificial intelligence inputs and files. However, cybersecurity and AI threats continue to evolve. For this reason, the project considers future improvements that will expand detection capabilities, improve analysis accuracy, and increase system scalability.

## New Threat Detection

As AI-based attacks become more sophisticated, the system can be extended to detect additional threat types, such as:

- Advanced prompt injection techniques

- New data poisoning strategies

- Hidden malicious model behaviors

- More complex steganography methods

Future versions may integrate new detection models and continuously updated threat databases to keep the system effective against emerging risks.

## Scoring Improvements

Currently, the system evaluates files using defined detection modules. In the future, the scoring system can be improved to provide more precise and informative risk assessments.

Possible improvements include:

- Weighted scoring models based on threat severity

- Machine learning–based risk evaluation

- More detailed threat classification

- Visual risk dashboards for users

These improvements would allow users to better understand the level and type of risk detected in analyzed files.

## Scalability

To support larger workloads and real-world deployment, the system can be improved to scale efficiently.

Future scalability improvements may include:

- Cloud-based infrastructure

- Microservices architecture

- Parallel processing for file analysis

- Container orchestration (e.g., Docker environments)

These enhancements would allow the platform to process a higher volume of files while maintaining performance and reliability.
