# How this project was developed

This project was developed using the Scrum methodology, assigning specific roles to manage the different parts of the project. This approach allowed the team to organize tasks efficiently and ensure that all components of the system were properly handled.

During the development process, we held 15-minute dailys to share progress updates, discuss current tasks, and identify any potential obstacles. The work was organized into sprints, which helped structure the development process and allowed the team to focus on completing specific features during each iteration.

To manage the project more effectively, we created a workspace in Jira, where tasks were organized according to the different features of the system. Each task was assigned to a team member to ensure clear responsibilities and maintain proper tracking of the project’s progress.

This structured workflow helped the team stay organized and contributed to completing the project in an efficient and coordinated manner.
