Guía de Inicio Rápido

  • 1. Crea una cuenta en PoisonAI o inicia sesión.
  • 2. Dirígete a la sección 'IA Scanner' desde el menú.
  • 3. Selecciona el tipo de archivo que deseas analizar: imagen, texto o código.
  • 4. Carga tu archivo y presiona 'Analizar'.
  • 5. Revisa el informe de seguridad con el puntaje de riesgo.

Tipos de Análisis

  • Imágenes: Detecta esteganografía, contenido oculto, metadata maliciosa y payloads embebidos en archivos de imagen (PNG, JPG, BMP, GIF).
  • Archivos de texto: Analiza archivos .txt en busca de URLs maliciosas, patrones de phishing, scripts ofuscados y contenido sospechoso.
  • Código fuente: Escanea código en busca de backdoors, inyecciones, vulnerabilidades conocidas y patrones de malware.

Sistema de Puntaje

  • 0–30: Seguro — No se detectaron amenazas. El archivo es confiable.
  • 31–69: Precaución — Se encontraron patrones sospechosos. Revisar manualmente.
  • 70–100: Peligroso — Alta probabilidad de contenido contaminado. No utilizar sin revisión profunda.

API Reference

  • POST /api/v1/scan — Envía un archivo para análisis. Acepta multipart/form-data.
  • GET /api/v1/report/:id — Obtiene el informe de un análisis previo.
  • GET /api/v1/history — Lista el historial de análisis del usuario autenticado.
  • Headers requeridos: Authorization: Bearer <token>

Especificaciones Técnicas

  • Motor de IA: Modelo de deep learning entrenado con +2M de muestras de malware.
  • Precisión: 97.3% de tasa de detección con 0.8% de falsos positivos.
  • Tiempo de respuesta: < 3 segundos para archivos de hasta 50MB.
  • Infraestructura: Procesamiento en la nube con encriptación end-to-end.

Información Legal

  • Términos de Uso: Al utilizar PoisonAI, aceptas que los archivos analizados se procesan de forma temporal y no se almacenan permanentemente en nuestros servidores.
  • Política de Privacidad: No compartimos datos de usuario con terceros. Toda la información es encriptada en tránsito y en reposo.
  • Cumplimiento: PoisonAI cumple con GDPR y regulaciones de protección de datos.
  • Responsabilidad: Los resultados del análisis son orientativos y no sustituyen una auditoría de seguridad profesional.
